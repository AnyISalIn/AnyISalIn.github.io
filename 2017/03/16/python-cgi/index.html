CGI 的一些问题

有几个比较关键的问题

  • 所有脚本都必须有执行权限
  • 不同操作系统的换行符可能会有问题
  • 要使用特定的扩展名去访问
  • cgi 脚本中必须指定解释器
  • 性能不行

在 Python 中解决这些问题可能就需要用到 WSGI 了
